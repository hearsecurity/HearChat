# Compilation Steps

## HearClient

<p>
 $ cd HearClient/src/
 $ javac *.java 
 $ jar cfe HearClient.jar HearClient *.class res/ <br>
 $ java -jar HearClient.jar </b></p>

## HearServer

<p>
 $ cd HearServer/src/
 $ javac *.java 
 $ jar cfe HearServer.jar HearServer HearServer.class <br>
 $ java -jar HearServer.jar </b></p>
